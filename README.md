# gs
Go-Spring 开发工具

```
gs pull spring-*/starter-*
gs push spring-*/starter-*
gs remove spring-*/starter-*
gs release tag
```

| Command   | Description                   |
| :-----    | :-----                        |
| pull      | 新建并拉取 go-spring 子项目      |
| push      | 将修改推送到 go-spring 子项目    |
| remove    | 本地移除 go-spring 子项目       |
| release   | 本地所有子项目发布远程标签        |
